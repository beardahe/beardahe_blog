# Lessons Learned

- 当前会话开始：暂无与本项目相关的历史纠错经验可复盘。
- 纠正记录（2026-02-28）：当用户反馈“页面打不开”时，先验证 dev server 是否运行，再给 URL；对访问类交付默认附带“启动命令 + 端口 + host 说明”。
- 纠正记录（2026-02-28）：当用户明确“可直接执行所有更改”时，默认连续完成整批实现与验证，不再分段等待确认；UI 需求明确时（如白色简约）优先统一全站主题样式。
- 纠正记录（2026-02-28）：当用户反馈“显示不正常”且问题为全站排版层级时，优先抽离全局主题文件并清除页面级样式分散，实现结构类名复用，避免继续在单页打补丁。
